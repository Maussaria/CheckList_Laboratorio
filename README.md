# 📋 Checklist Diário - Laboratório II (Digital)

Aplicação web para gerenciamento do checklist diário do Laboratório III do SENAC Linhares.

## 🚀 Características

- ✅ Preenchimento online do checklist diário (Segunda a Sexta)
- 📄 Geração de PDF completo ao final da semana
- 💾 Armazenamento temporário dos dados durante a semana
- 📱 Interface responsiva e moderna
- 🌓 Suporte a tema claro e escuro
- 🔄 Navegação por abas para cada dia da semana

## 📝 Como Usar

### Preenchimento Semanal

1. **Configure as informações da semana:**
   - Nome do responsável
   - Período (data início e fim)
   - Líder de turma

2. **Preencha o checklist diariamente:**
   - Use as abas para navegar entre os dias (Segunda a Sexta)
   - Marque os checkboxes dos itens realizados
   - Adicione observações quando necessário

3. **Registre problemas:**
   - Role até a seção "Problemas Reportados ao Professor"
   - Anote qualquer problema encontrado em cada dia

### Geração do PDF

1. Ao final da semana, clique no botão **"📄 Gerar PDF Semanal"**
2. O PDF será gerado automaticamente com todos os dados da semana
3. Salve o PDF no seu computador
4. Faça upload para o OneDrive

### Limpar Dados

- Para iniciar uma nova semana, clique em **"🔄 Limpar Tudo"**
- Isso apagará todos os dados e permitirá começar do zero

## 📂 Estrutura do Checklist

### ✅ Abertura do Laboratório (07:30)
- Ligar ar-condicionado
- Acender luzes
- Ligar todos os 25 PCs
- Verificar boot correto

### 🖥️ Equipamentos
- Monitores funcionando
- Teclados funcionando
- Mouses funcionando
- Caixa de som OK
- Datashow OK (se usar)

- Desligar monitores

## 🔄 Workflow Recomendado

1. **Segunda-feira (manhã):** Preencher dados da semana e checklist do dia
2. **Terça a Quinta:** Continuar preenchendo diariamente
3. **Sexta-feira (final):** Completar o último dia e gerar o PDF
4. **Fazer upload do PDF para o OneDrive**
5. **Limpar o formulário para a próxima semana**

## 💡 Dicas

- **Backup:** Gere o PDF ao final de cada dia como backup adicional
- **Observações:** Use o campo de observações para anotar números de PCs com problema
- **Mobile:** A aplicação funciona perfeitamente em tablets e smartphones
- **Offline:** Os dados ficam salvos durante a semana mesmo se fechar o navegador

## ⚠️ Importante

- **Dados Temporários:** Os dados são armazenados apenas durante a sessão
- **PDF Semanal:** Sempre gere e salve o PDF antes de limpar os dados
- **OneDrive:** Organize os PDFs por data/semana no OneDrive

## 🆘 Problemas Comuns

### PDF não está baixando
- Verifique se o navegador não está bloqueando downloads
- Tente usar outro navegador (Chrome, Firefox, Edge)

### Dados foram perdidos
- Os dados são temporários e só persistem durante a sessão
- Sempre gere o PDF antes de fechar o navegador

### Aplicação não carrega
- Verifique se o GitHub Pages está ativo
- Aguarde alguns minutos após ativar o GitHub Pages
- Limpe o cache do navegador (Ctrl + F5)
